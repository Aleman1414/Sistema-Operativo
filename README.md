# Sistema Operativo Simulado

Este proyecto es una simulación de un sistema operativo desarrollado en Python para fines educativos en la clase de Sistemas Operativos.

## Características

- **Interfaz Gráfica de Usuario**: Utiliza Tkinter para una GUI intuitiva.
- **Administración de Recursos**: Monitorea CPU, memoria, almacenamiento y periféricos.
- **Administración de Archivos**: Permite crear, borrar, copiar y listar archivos simulados.
- **Administración de Tareas**: Lista y permite terminar procesos del sistema.

## Requisitos

- Python 3.x
- psutil (instalado vía pip)

## Instalación

1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `python main.py`

## Uso

La aplicación se divide en tres pestañas principales correspondientes a cada módulo de administración.
